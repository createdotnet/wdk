This conditional tag will check whether a website user is logged in.

### Example
~~~
	<!--WDK:if:plugin:useraccount:isloggedin-->
		<p>You are logged in</p>
		<!--WDK:else-->
		<p>You are not logged in</p>
	<!--WDK:endif-->
~~~
