## Tag

`<!--WDK:footer:end-->`

## Description

This is a required tag that ends the footer section of your template.
For more information, please see our footer turorial here.

## Example

~~~
<!--WDK:header:start-->
<html>
  <head></head>
  <body>
    <!--WDK:header:end-->

      Dynamic page content is automatically inserted here.

    <!--WDK:footer:start-->
    <!--WDK:display:sitefooter-->
  </body>
</html>
<!--WDK:footer:end-->
~~~
