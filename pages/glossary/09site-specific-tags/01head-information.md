##Tag

`<!--WDK:display:headinfo-->`

##Description

This tag returns generated HTML `<title>` and `<meta>` tags for use within the `<head>` section of the document.

##Example

~~~
<head>
  <!--WDK:display:headinfo-->
</head>
~~~
